CS 426 Senior Research Project

UNR Computer Science & Engineering Department
Spring 2019
Team Number: 16

Project Description

The Pythia project aims to implement a verbal interface for NAASIC’s self-driving car. The team, consisting of Sybille Horcholle, Nathan Thom, and Bryson Lingenfelter, will implement a robust natural language processor to work in tandem with the vehicle’s pre-existing systems. This interface will provide the user with the ability to give commands and queries to the vehicle, have the vehicle respond in a meaningful way, and allow for bi-directional communication during high-risk maneuvers such as unprotected left-hand turns or lane changes.
